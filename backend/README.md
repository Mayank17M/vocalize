# Setting up 

### How to Install IPFS

Download [IPFS](https://ipfs.io/docs/install/)

Start an IPFS node with your CLI

```
ipfs init 
```

```
ipfs daemon 
```

Access your file at: https://gateway.ipfs.io/ipfs/(your_IPFS_hash)

To get all available test account
```
node_modules/.bin/testrpc
```
